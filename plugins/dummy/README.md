# Dummy Template Plugin

A template plugin demonstrating:
- **SVG Icons**: Defined directly in `plugin.json`
- **Dynamic Settings**: Configurable via the Settings page
- **UI Updates**: Reacting to setting changes

## Setup

1. Enable the plugin in Nova.
2. Go to Settings and find "Dummy Template" to change the greeting or background color.
3. See the changes reflected in the plugin page.
